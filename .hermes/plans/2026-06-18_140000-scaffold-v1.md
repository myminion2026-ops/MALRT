# MALRT v0.1.0 Scaffold Implementation Plan

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Scaffold MALRT with a FastAPI backend + web dashboard for submitting indicators and monitoring provider submissions.

**Architecture:** FastAPI serves both the REST API and a single-page dashboard. SQLite tracks submissions. Reporters are pluggable modules. SSE streams live updates to the dashboard.

**Tech Stack:** Python 3.11+, FastAPI, SQLite (via aiosqlite), Jinja2, vanilla JS (no build step), uvicorn.
